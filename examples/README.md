# JTOX Examples

Working examples demonstrating JTOX features.

## Examples

### 01 — Hero Banner
A single section with settings-bound text, conditional CTA button, and style settings.
Demonstrates: `section`, `heading`, `text`, `link`, `conditional`, `$bind`, settings schema.

### 02 — Product Grid
A product listing section with collection iteration, component references, and empty states.
Demonstrates: `each`, `component` ($ref), `conditional` (nested), `fallback`, `empty`,
`datasource` setting, props passing.

### 03 — Full Page
A complete page with layout, header, navigation, and page template.
Demonstrates: `layout`, `page`, `slot`, `each` (navigation), `conditional` (logo fallback),
`context` scope, `theme` scope, `allowedSections`, `sectionRules`, action attributes.

### 04 — Complete Theme
A minimal but complete theme package with theme.json, sections, components, layouts, pages,
headers, footers, settings, and props interfaces.
Demonstrates: Full theme file structure, presets, setting groups, composition hierarchy.
